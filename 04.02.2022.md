## Задача №3739. Первое и последнее вхождение
```
s = input()
a1 = s.find('f')
a2 = s.rfind('f')
if a1 == -1:
    print()
elif a1 == a2:
    print(a1)
else:
    print(a1, a2)
```
## Задача №3740. Второе вхождение
```
a = input()
count = 0
for i in range(len(a)):
    if a[i] == "f":
        count += 1
        if count == 2:
            print(i)  
if count == 1:
        print('-1')
elif "f" not in a:
    print('-2')
```

## №3742. Обращение фрагмента
```

a2 = s[s.find('h'):s.rfind('h') + 1]

print(a1 + a2[::-1] + a3)
```
