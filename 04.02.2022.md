## Задача №3739. Первое и последнее вхождение
```
s = input()
a = s.find('f')
b = s.rfind('f')
if a == -1:
    print()
elif a == b:
    print(a)
else:
    print(a, b)
```
## Задача №3740. Второе вхождение
```
a = input()
count = 0
for i in range(len(a)):
    if a[i] == "f":
        count += 1
        if count == 2:
            print(i)  
if count == 1:
        print('-1')
elif "f" not in a:
    print('-2')
```
